- Nexus — это лаунчер Minecraft, основанный на [PojavLauncher](https://github.com/PojavLauncherTeam/PojavLauncher), который запускает [Minecraft: Java Edition](https://www.minecraft.net/) на устройствах Android.
- Nexus стремится улучшить пользовательский опыт, переработав интерфейс приложения, добавив больше практичных функций и снизив порог входа, делая Minecraft доступным для большего числа игроков.

> [!WARNING]
> Внимание! Этот проект является лишь **переработкой сообществом**, так же как PojavLauncher разработан на основе [Boardwalk](https://github.com/zhuowei/Boardwalk).

<h2 align="center">Новые функции</h2>

- [x] Переработанный макет приложения для улучшения эстетики и удобства использования.
- [x] Встроенный простой файловый менеджер, чтобы не беспокоиться об ограничениях прав доступа Android!
- [x] Улучшенное управление версиями, настраиваемое расположение игровой директории, поддержка внешнего хранилища.
- [x] Добавлены новые рендереры с поддержкой плагинов рендеринга!
- [x] Загрузка модов, модпаков, ресурспаков, сохранений и шейдеров прямо внутри лаунчера!
- [x] Настраиваемый значок виртуальной мыши и фоновое изображение лаунчера.
- [x] Новые цвета интерфейса.
- [x] И многое другое!
